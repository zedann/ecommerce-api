# CATEGORIES

### FIELDS

- name
- slug

## END POINTS

#### GET /api/v1/categories

Get all categories

#### GET /api/v1/categories/:id

Get category by id

#### POST /api/v1/categories

Create category

**Request Body:**

```json
{
  "name": "example"
}
```

#### PATCH /api/v1/categories/:id

Update category

**Request Body:**

```json
{
  "name": "example"
}
```

#### DELETE /api/v1/categories/:id

Delete category
